## Project Description: 
The Outfit Planner is a simple fashion management web application. The goal of the project is to help users store and organize clothing combinations in one place.

The project demonstrates the use of Flask web development, Object-Oriented Programming(OOP), file handling with JSON, modular programming and exception handling.

## Features: 
Add new outfits; 
Upload outfit images; 
View all saved outfits; 
Edit outfit information; 
Delete outfits; 
Filter outfits by: style, weather and event; 
Outfit recommendations; 
JSON file storage; 
Responsive and modern user interface.

## Technologies Used: 
Python; 
Flask; 
HTML; 
CSS; 
JSON; 
Werkzeug.

## Installation Instructions: 

1. Clone the repository 
git clone <repository-link>

2. Open the project folder 
cd outfit-planner

4. Install dependencies 
pip install -r requirements.txt

## How to Run the Project: 
Run the following command: python main.py
Then open the browser and go to: http://127.0.0.1:5000
